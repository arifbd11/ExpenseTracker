ExpenseTracker Build Instructions

Changed made

Prerequisites:
- JDK installed
- Maven installed
- Git installed

Steps to Build and Run ExpenseTracker:

1. Clone the ExpenseTracker repository from GitHub.
2. Navigate to the ExpenseTracker directory.
3. Run the build script:
   $ sh build.sh
4. Upon successful build, deploy the application using the generated package.

For any issues or assistance, contact the development team.
